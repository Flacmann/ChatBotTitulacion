#  ChatBotAdmProy – Chatbot de Administración de Proyectos

Este proyecto forma parte de mi trabajo de titulación.  
Yo Diego Noel Serrano Quiroz Desarrollé un **chatbot educativo** para Telegram que responde preguntas sobre **Administración de Proyectos**, con base en la bibliografía de Kerzner, Meredith, Rivera y el PMI (PMBOK).  

El bot puede responder dudas sobre los **procesos, áreas de conocimiento, gestión de riesgos, calidad, costos, integración, entre otros temas** del curso.

---

##  Descripción general
El bot fue programado en **Python** utilizando **Flask** para manejar el servidor y las solicitudes del webhook de Telegram.  
El conocimiento está almacenado en un archivo JSON (`temas_completos_formato2.json`), que contiene todas las preguntas y respuestas organizadas por tema.

El objetivo es ofrecer una herramienta sencilla y práctica para que los estudiantes consulten información de administración de proyectos desde Telegram.

---

##  Requisitos
- Python 3.10 o superior  
- Cuenta gratuita en [Deta Space](https://www.deta.sh)  
- Token de bot generado con [@BotFather](https://t.me/BotFather) en Telegram
